## Requirements

The following tools are required in order to start the installation.

- [VirtualBox](https://www.virtualbox.org/)
- [Vagrant](https://www.vagrantup.com/)

## Installation


1. Clone this repository
2. Run `composer start`
4. Run `vagrant up`
